# Quiz Platform

A full-stack quiz platform where users can participate in quizzes, track progress, and view performance analytics.
The application provides a smooth quiz-taking experience with authentication, dynamic question navigation, and progress tracking.

## Live Demo

https://splendorous-hamster-e3fb28.netlify.app

## Tech Stack

### Frontend

- React.js
- JavaScript (ES6+)
- CSS Modules
- React Router
- Bootstrap

### Backend

- Node.js
- Express.js
- REST API
- JWT Authentication

### Database

- MongoDB

### Other Tools

- Axios
- Chart.js / Data visualization

## Installation

### Clone the repository

- git clone https://github.com/SathappanRamesh/quiz-site.git

- Go to the project directory

- cd quiz-site

## Install dependencies

### Frontend

- cd client
- npm install

### Backend

- cd server
- npm install

## Running the Application

### Start the frontend

- cd my-proj
- npm start

### Start the backend server

- cd server
- npm run dev

## Environment Variables

### Create a .env file in the server directory and add the following:

- MONGO_URI=your_mongodb_connection
- JWT_SECRET=your_jwt_secret
- PORT=3000
